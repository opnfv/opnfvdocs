Introduction
---------------

The OPNFV Security Guide is the collaborative work of many individuals, involved
 in both the OPNFV Security Group and the wider OPNFV community.

The purpose of this guide is to provide the best practice security guidelines
for deploying the OPNFV platform. It is a living document that is updated as new
changes are merged into it's repository.
